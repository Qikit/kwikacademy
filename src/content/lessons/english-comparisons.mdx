## 1. Зачем уметь сравнивать

Половина оценочных высказываний — это сравнения. «Эта квартира дороже», «он бегает быстрее меня», «сегодня холоднее, чем вчера», «самый длинный отчёт за месяц». Без сравнительной и превосходной степени остаётся либо сухая констатация («это дорого»), либо клишированные обороты вроде «more than expected», за которыми не видно сути.

В русском у нас тоже два способа сделать степень: суффикс («длиннее», «теплее») и аналитика («более длинный», «более тёплый»). В английском та же логика: короткие прилагательные берут суффикс `-er / -est`, длинные — конструкцию `more / the most`. Проблема в том, что русскоязычные регулярно ошибаются в трёх местах: смешивают оба способа («more better»), забывают артикль `the` перед превосходной («best player» вместо «the best player») и не различают `farther / further`, `less / fewer`, `as ... as / -er than`.

Этот урок даёт правила выбора и орфографии так, чтобы форма приходила автоматически, и закрывает типовые ошибки, по которым русскоязычного слышно сразу.

## 2. Главное правило

<Callout type="purple" title="Ядро темы">
Сколько слогов в прилагательном — столько и решает. **1 слог** (и часть 2-сложных на `-y`) → `-er / the -est`. **3 слога и больше** → `more / the most`. **Перед превосходной — всегда `the`**. Никогда не оба способа сразу: либо `bigger`, либо `more big` (последнее — неверно для big), но не `more bigger`.
</Callout>

## 3. Формы и грамматика

### Сравнительная степень

Используется, чтобы сказать «X больше / меньше / лучше, чем Y». Структура: `X is [comparative] than Y`.

| Тип прилагательного | Как образуется | Пример |
| --- | --- | --- |
| 1 слог | adj + `-er` | tall → taller, fast → faster |
| 2 слога, окончание `-y` | adj без `-y` + `-ier` | happy → happier, easy → easier |
| 2 слога (большинство) | `more` + adj | careful → more careful |
| 3 слога и больше | `more` + adj | interesting → more interesting, expensive → more expensive |

После сравнительной — `than`: `She is taller than me. The new flat is more expensive than the old one.`

### Превосходная степень

Используется, чтобы сказать «самый / наиболее ... из группы». Структура: `the [superlative] (in / of ...)`.

| Тип прилагательного | Как образуется | Пример |
| --- | --- | --- |
| 1 слог | `the` + adj + `-est` | the tallest, the fastest |
| 2 слога, окончание `-y` | `the` + adj без `-y` + `-iest` | the happiest, the easiest |
| 2 слога (большинство) | `the most` + adj | the most careful |
| 3 слога и больше | `the most` + adj | the most interesting, the most expensive |

Артикль `the` обязателен: «He is the best player», не «He is best player». Исключения — после притяжательного местоимения (`my best friend`) или после `be` в значении «лучшее из всего» (`Honesty is best`, идиоматично).

### Орфография -er / -est

| Правило | Положительная | Сравнительная | Превосходная |
| --- | --- | --- | --- |
| Молчаливое `-e` исчезает | late | later | the latest |
| То же | nice | nicer | the nicest |
| Короткий CVC удваивает согласную | big | bigger | the biggest |
| То же | thin | thinner | the thinnest |
| То же | hot | hotter | the hottest |
| `-y` после согласной → `-ier / -iest` | happy | happier | the happiest |
| То же | easy | easier | the easiest |
| `-y` после гласной — остаётся | grey | greyer | the greyest |
| Обычное добавление | tall | taller | the tallest |
| То же | cheap | cheaper | the cheapest |

CVC = согласная + гласная + согласная в конце слова под ударением. `big` (b-i-g) — удваиваем. `cool` (c-oo-l) — две гласные, не удваиваем. `cold` (c-o-l-d) — заканчивается на две согласные, не удваиваем.

### Неправильные степени

Эти формы надо запомнить — правила к ним не применяются.

| Положительная | Сравнительная | Превосходная |
| --- | --- | --- |
| good | better | the best |
| bad | worse | the worst |
| far | farther / further | the farthest / the furthest |
| little | less | the least |
| much / many | more | the most |
| old (для членов семьи) | elder | the eldest |

`elder / eldest` — только перед существительным и только про старшинство в семье: `my elder brother`. В остальных случаях — `older / oldest`.

### Наречия

- Короткие наречия (`fast`, `hard`, `early`, `late`, `soon`) ведут себя как короткие прилагательные: `faster, the fastest, harder, the hardest`.
- Наречия на `-ly` (`carefully`, `quickly`, `slowly`) — через `more / the most`: `more carefully, the most carefully`.
- Исключение: `well → better → the best` (как у прилагательного `good`).
- `badly → worse → the worst`.

`He runs faster than me. She speaks more clearly than I do. The team finished the task the most efficiently of all departments.`

### Конструкции

- **`as ... as`** — равенство: `She is as tall as her sister.` Между as — основная форма, не сравнительная.
- **`not as ... as` / `not so ... as`** — неравенство в сторону меньше: `He's not as fast as her.`
- **`less ... than`** — меньше по качеству: `It's less interesting than I thought.` Работает с длинными прилагательными.
- **`the + comparative, the + comparative`** — двойная сравнительная: «чем ..., тем ...». `The more, the merrier. The harder you work, the better the result.`
- **Модификаторы при сравнительной**: `much`, `a lot`, `far`, `slightly`, `a bit`. Идут перед сравнительной формой. `Much faster. A bit better. Far more interesting.` Нельзя `very faster` — `very` сочетается только с основной формой (`very fast`).

## 9. Мнемоника

<Callout type="teal" title="Лайфхак выбора способа">
1 слог → всегда `-er / -est`. 3 слога и больше → всегда `more / the most`. 2 слога — смотри окончание: на `-y` → `-ier / -iest` (happy → happier), всё остальное (`careful, modern, useful`) → `more / the most`. На спорные двусложные (`quiet`, `clever`, `simple`) допустимы оба варианта — выбирай короткий, если сомневаешься.
</Callout>

## 10. Логика расхождения

<Callout type="blue" title="Почему the обязателен с превосходной">
Превосходная степень по смыслу указывает на единственный объект из группы: «самый высокий» = ровно один. Это и есть классическая работа определённого артикля — выделить конкретный, уже известный из контекста элемент. Поэтому `the best player` — единственная грамматическая форма: артикль не украшение, а часть конструкции. Сравнительная же говорит «больше, чем Y», то есть может относиться к множеству объектов («любой из этих ноутбуков быстрее старого»), и определённости там нет — артикль не нужен. Исключения вроде `my best friend` работают потому, что притяжательное местоимение само по себе уже даёт определённость и забирает роль артикля.
</Callout>

## 12. Типовые ошибки русскоязычных

| Неверно | Верно | Почему |
| --- | --- | --- |
| She is more better than him | She is better than him | Двойная сравнительная: либо `better`, либо `more good` (последнее тоже неверно, нужно `better`). Никогда оба сразу. |
| He is best player on the team | He is the best player on the team | Перед превосходной — обязательный `the`. |
| This film is expensiver | This film is more expensive | `expensive` — 3 слога, через `more`. Окончание `-er` тут не работает. |
| Happyer / happyest | Happier / happiest | После согласной `-y` превращается в `-i` перед суффиксом. |
| Bigest / runing | Biggest / running | CVC под ударением → удваиваем согласную перед `-est / -ing`. |
| I have less books than you | I have fewer books than you | `less` — для неисчисляемых (time, money). С исчисляемыми — `fewer`. |
| He runs good | He runs well | После глагола действия — наречие `well`, не прилагательное `good`. |
| She is as faster as him | She is as fast as him | Между `as ... as` идёт основная форма, не сравнительная. |
| He's taller than I am taller | He's taller than I am / He's taller than me | Во второй части сравнения сравнительная форма не повторяется. |
| Better than then | Better than than | (опечатка примера) — путаница `than` (для сравнения) и `then` (затем). `Better than that. Then we left.` |
| I need more further information | I need further information | `further` уже несёт смысл «дополнительный», `more` тут лишнее. |
| The most longest movie | The longest movie | Тоже двойная превосходная: либо `the longest`, либо `the most long` (последнее неверно для `long`). |
| She is very faster than him | She is much faster than him | `very` сочетается только с основной формой (`very fast`). С сравнительной — `much / far / a lot`. |
| My older brother (если про старшинство в семье) | My elder brother | Для старшинства внутри семьи — `elder / eldest`. В остальных случаях `older / oldest` нормально. |

## 13. Шпаргалка

**Орфография -er / -est:**

| Окончание / тип | Правило | Пример |
| --- | --- | --- |
| Обычное 1-сложное | + -er / -est | tall → taller → the tallest |
| Молчаливое -e | + -r / -st | late → later → the latest |
| CVC под ударением | удвоить согласную | big → bigger → the biggest |
| -y после согласной | -y → -i + -er / -est | happy → happier → the happiest |
| -y после гласной | оставить -y | grey → greyer → the greyest |
| 2 слога (большинство) | more / the most | careful → more careful → the most careful |
| 3+ слогов | more / the most | expensive → more expensive → the most expensive |

**Неправильные:**

| Положительная | Сравнительная | Превосходная |
| --- | --- | --- |
| good / well | better | the best |
| bad / badly | worse | the worst |
| far | farther (расстояние) / further (доп.) | the farthest / the furthest |
| little | less | the least |
| much / many | more | the most |
| old (семья) | elder | the eldest |

**Конструкции:** `X is -er than Y` · `X is the -est in/of Z` · `as ... as` · `not as / not so ... as` · `less ... than` · `the more, the merrier` · `much / a bit / far / slightly + comparative`.
