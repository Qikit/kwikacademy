## 1. Зачем пассив

В русском пассив есть, но звучит он гибко: «строится дом», «дом был построен», «дом строят» — три формы под одну ситуацию, и часть из них в живой речи легко превращается в актив без потери смысла («они строят дом»). В английском пассив устроен жёстче: одна базовая конструкция `be + V3` во всех временах, и её грамматическая дисциплина не прощает приблизительности.

Без пассива не получится написать ни одного нормального формального текста. Инструкции к технике, новости, отчёты, научные статьи, описания процессов, юридические документы, политики компаний — там пассив базовая ткань языка. «The form must be signed before submission.», «The package was delivered yesterday.», «The bug has been fixed.» — выкинь пассив, придётся придумывать активного героя, которого в реальности нет, или который не важен.

Перепутаешь форму — фраза не сломается, но прозвучит коряво или сменит смысл. «The cake was baking when I arrived» (пирог сам пёкся) ≠ «The cake was being baked when I arrived» (его пекли, когда я пришёл). Этот урок про то, как собирать пассив осознанно во всех временах, как решать, оставлять ли `by + agent`, и где русскоязычные регулярно спотыкаются.

## 2. Главное правило

<Callout type="purple" title="Ядро темы">
Пассив — это **be + V3** (третья форма глагола, past participle). Время передаётся формой `be`: `is done` (Present Simple), `was done` (Past Simple), `is being done` (Present Continuous), `has been done` (Present Perfect), `will be done` (Future Simple), `must be done` (модальный пассив). Смысловой глагол всегда стоит в V3 и не меняется.
</Callout>

## 3. Формы и грамматика

### Преобразование Active → Passive

Три шага:

1. **Объект** активного предложения становится **подлежащим** пассивного.
2. **Глагол** превращается в `be + V3` в том же времени.
3. **Подлежащее** активного либо уходит в `by + agent`, либо опускается.

`Shakespeare wrote the play.` → `The play was written by Shakespeare.`

Объект *the play* стал подлежащим. Глагол *wrote* (Past Simple) → *was written* (be в Past Simple + V3). Подлежащее *Shakespeare* ушло в *by Shakespeare* — здесь оно важно, поэтому остаётся.

### Пассив во всех временах

| Время | Активная формула | Пассивная формула | Пример пассива |
| --- | --- | --- | --- |
| Present Simple | V / V-s | am / is / are + V3 | English **is spoken** here. |
| Past Simple | V2 | was / were + V3 | The bridge **was built** in 1990. |
| Present Continuous | am / is / are + V-ing | am / is / are **being** + V3 | The road **is being repaired**. |
| Past Continuous | was / were + V-ing | was / were **being** + V3 | The cake **was being baked** when I arrived. |
| Present Perfect | have / has + V3 | have / has **been** + V3 | The report **has been finished**. |
| Past Perfect | had + V3 | had **been** + V3 | The letter **had been sent** before I called. |
| Future Simple | will + V | will **be** + V3 | The work **will be done** tomorrow. |
| Future Perfect | will have + V3 | will have **been** + V3 | The project **will have been completed** by Friday. |
| be going to | am / is / are going to + V | am / is / are going to **be** + V3 | The decision **is going to be made** soon. |

Future Continuous (`will be being done`) теоретически существует, но в живом языке избегается из-за уродливого «be being». Если очень нужно — обычно перефразируют активом или Future Continuous активом с другим подлежащим.

### Модальный пассив

После любого модального — `be + V3`. Сам модальный не меняется, время передаётся контекстом или модальной формой прошлого (`could have been done`, `should have been done` — тема урока про conditionals).

- `must be done` — должно быть сделано.
- `should be checked` — стоит проверить (буквально «должно быть проверено»).
- `can be solved` — может быть решено.
- `could be improved` — могло бы быть улучшено.
- `may be delayed` — может быть отложено.
- `might be cancelled` — может быть отменено.
- `has to be approved` — нужно одобрить.

### Отрицание и вопрос

Отрицание и вопрос строятся по тем же правилам, что и в активе соответствующего времени — потому что `be` (или вспомогательный модальный) сам по себе является auxiliary.

- `The report wasn't finished on time.` — Past Simple passive, отрицание через was + not.
- `Has the report been finished?` — Present Perfect passive, вопрос через инверсию has и подлежащего.
- `Will the work be done tomorrow?` — Future Simple passive, инверсия will.
- `Must this form be signed?` — модальный пассив, инверсия модального.

### Пассив с двумя дополнениями

Глаголы `give`, `send`, `tell`, `offer`, `show`, `teach`, `lend`, `pay`, `promise`, `bring`, `award` имеют два дополнения: получателя и вещь. Из такой активной фразы получается **два варианта пассива** — какое из двух дополнений станет подлежащим.

<RuleList rules={[
  "<strong>Получатель как подлежащее — обычно предпочтительнее в английском.</strong> «She gave me a book» → <em>I was given a book.</em> Так чаще говорят, особенно в разговорной речи.",
  "<strong>Вещь как подлежащее — возможно, но звучит формальнее.</strong> «She gave me a book» → <em>A book was given to me.</em> Получатель уходит в to me. Обычно используется в формальных текстах или когда вещь важнее.",
  "<strong>В русском «мне дали книгу» — это калька неработающая.</strong> Нельзя сказать «To me was given a book». Правильно: <em>I was given a book.</em> — получатель = подлежащее.",
  "<strong>Если вещь важна сама по себе — её можно вынести в начало.</strong> <em>The prize was awarded to three winners. The keys were handed to the owner.</em>",
]} />

## 9. Мнемоника

<Callout type="teal" title="Сборка пассива за три шага">
Чтобы собрать пассив, иди в порядке: **время → форма be → V3**. Сначала определи время предложения (это решает, в какой форме будет `be`: is/are, was/were, has been, will be и т.д.). Потом ставь `be` в нужной форме. Потом — V3 смыслового глагола. Если глагол неправильный — V3 берёшь из таблицы (write → written, give → given, take → taken, do → done). Это всё. Никаких изменений основного глагола, никаких -ing там, где их быть не должно.
</Callout>

## 10. Логика расхождения

<Callout type="blue" title="Почему английский так дисциплинирован в пассиве">
У английского глагола исторически развилась чёткая трёхчастная система: V1 (do), V2 (did), V3 (done). V3 одновременно работает как причастие — отсюда и `have done` (Perfect), и `be done` (Passive). Эта симметрия удобна для языка: одна форма участвует в двух конструкциях, нагрузка распределена. Русский пошёл другим путём: возвратная форма на -ся, краткое причастие, безличный оборот — три способа выразить то, что в английском упаковано в одну формулу `be + V3`. Поэтому английский пассив звучит «механически точно», а русский — «гибко». Злоупотребление пассивом в английском — известный стилистический грех (его критикуют все руководства по стилю), потому что он утяжеляет текст и убирает живого героя. Используй пассив, когда герой не важен или не нужен, — и возвращайся в актив, когда есть кого назвать.
</Callout>

## 12. Типовые ошибки русскоязычных

| Неверно | Верно | Почему |
| --- | --- | --- |
| The report was finish yesterday | The report was finished yesterday | После be — V3, не V1. finish → finished (правильный глагол). |
| The letter was writed yesterday | The letter was written yesterday | write — неправильный глагол. V3 = written, не writed. См. урок про прошедшее. |
| The cake is baking right now (имелся в виду процесс пассива) | The cake is being baked right now | Present Continuous passive = is/are + **being** + V3. Без being получается актив с другим смыслом. |
| The report have been finished | The report has been finished | The report — единственное число → has, не have. Согласование берётся у подлежащего пассива. |
| The bug is fixed by us yesterday | The bug was fixed by us yesterday | Маркер «yesterday» = прошлое → was, не is. Be ставится в нужное время. |
| It was happened in 2020 | It happened in 2020 | happen — непереходный глагол. Пассив с ним невозможен. Только актив. |
| To me was given a book | I was given a book | Получатель становится подлежащим: I was given. Калька «to me was» — неработающая. |
| English is speak here | English is spoken here | После be — V3 (spoken), не V1 (speak). speak → spoke → spoken. |
| The form must to be signed | The form must be signed | После модального — be (без to), потом V3. Must to — двойная ошибка: модальный + to + be. |
| The window was broken with a thief | The window was broken by a thief | by — для исполнителя (живой человек / организация). with — для инструмента (with a hammer). |
| It is being being done | It is being done | Continuous passive = is + being + V3. Two being подряд — невозможно. |
| The road is repaired now (имелся в виду процесс) | The road is being repaired now | «Now» + процесс → Present Continuous passive (is being repaired). Is repaired = статичный факт. |

## 13. Шпаргалка

| Время | Формула пассива | Пример |
| --- | --- | --- |
| Present Simple | am / is / are + V3 | English is spoken here. |
| Past Simple | was / were + V3 | The bridge was built in 1990. |
| Present Continuous | am / is / are being + V3 | The road is being repaired. |
| Past Continuous | was / were being + V3 | The cake was being baked. |
| Present Perfect | have / has been + V3 | The report has been finished. |
| Past Perfect | had been + V3 | The letter had been sent. |
| Future Simple | will be + V3 | The work will be done tomorrow. |
| Future Perfect | will have been + V3 | The project will have been completed by Friday. |
| be going to | am / is / are going to be + V3 | The decision is going to be made. |
| Modal passive | modal + be + V3 | This form must be signed. |
| Modal passive (past hint) | modal + have been + V3 | It should have been checked. |
| Negative / Question | строится у формы be | Has the report been finished? — The report wasn't finished. |
| By + agent | оставлять, если важно | The play was written by Shakespeare. |
| With + instrument | для орудия, не исполнителя | The window was broken with a hammer. |
| Двойное дополнение | получатель → subject | I was given a book. |
| Непереходные глаголы | пассив невозможен | It happened. (не «It was happened».) |
