## 1. Зачем косвенная речь

Прямая речь — это кавычки и точная цитата: «She said, „I am tired“». Косвенная речь (reported speech) — пересказ: «She said she was tired». Без неё нельзя обсуждать диалоги, рассказывать, что сказал коллега, передавать новости, писать отчёты о встречах. Любой пересказ разговора — это reported speech.

В русском пересказ устроен почти как прямая речь: «Он сказал, что устал», «Она спросила, где я живу». Времена согласуются мягко: можно сказать «Он сказал, что устаёт» (настоящее в придаточном после прошедшего в главном), и никто не споткнётся. Английский так не работает. Если в главном предложении прошедшее время (`said`, `asked`, `told`), то и придаточное чаще всего сдвигается «на шаг назад» во времени — это **backshift**, согласование времён. Плюс меняются лица, указатели времени и места.

Перепутать — фраза не сломается, поймут. Но звучит ломано. «He said he is tired» вместо «He said he was tired» сразу выдаёт нерод­ного. А в вопросах русскоязычные регулярно ломают конструкцию: «He asked where did I live» — калька с английского прямого вопроса, в reported так нельзя. Этот урок про три кита косвенной речи: что сдвигается (backshift), что заменяется (лица, обстоятельства), и какие три формы есть у самих конструкций (statements, questions, imperatives).

## 2. Главное правило

<Callout type="purple" title="Ядро темы">
При пересказе после reporting verb в прошедшем времени (`said`, `told`, `asked`) всё в придаточном сдвигается **«на шаг назад» во времени**: Present → Past, Past → Past Perfect, will → would, can → could, must → had to. Параллельно меняются **лица** (I → he/she, my → his/her), **обстоятельства времени** (today → that day, tomorrow → the next day, now → then) и **места** (here → there, this → that). У вопросов и приказов — отдельные шаблоны: yes/no question через `if/whether` с прямым порядком, wh-question с прямым порядком, приказ через `told/asked + obj + to + V`.
</Callout>

## 3. Формы и грамматика

### Таблица backshift — главное

| Прямая речь (в кавычках) | Reported (после said) |
| --- | --- |
| Present Simple — *«I work»* | Past Simple — *said he worked* |
| Present Continuous — *«I am working»* | Past Continuous — *said he was working* |
| Present Perfect — *«I have worked»* | Past Perfect — *said he had worked* |
| Present Perfect Continuous — *«I have been working»* | Past Perfect Continuous — *said he had been working* |
| Past Simple — *«I worked»* | Past Perfect — *said he had worked* |
| Past Continuous — *«I was working»* | Past Perfect Continuous — *said he had been working* |
| Past Perfect — *«I had worked»* | Past Perfect (не сдвигается дальше) — *said he had worked* |
| Future `will` — *«I will come»* | `would` — *said he would come* |
| Future `going to` — *«I am going to come»* | *was/were going to* — *said he was going to come* |
| `can` — *«I can swim»* | `could` — *said he could swim* |
| `may` (возможность) — *«It may rain»* | `might` — *said it might rain* |
| `must` (обязанность) — *«I must go»* | `had to` — *said he had to go* |
| `must` (дедукция) — *«He must be tired»* | `must` (не сдвигается) — *said he must be tired* |
| `shall` — *«I shall help»* | `should / would` — *said he would help* |
| `could / would / might / should / ought to` | без изменений — *said he could / would / might do it* |

### Указатели времени и места

| Прямая | Reported |
| --- | --- |
| today | that day |
| tonight | that night |
| this morning | that morning |
| yesterday | the day before / the previous day |
| tomorrow | the next day / the following day |
| last week | the previous week / the week before |
| next month | the following month / the next month |
| now | then |
| ago | before / earlier |
| here | there |
| this (book) | that (book) |
| these (keys) | those (keys) |

### Лица и притяжательные

Меняются по логике пересказа. *He said, «I lost my keys»* → *He said he had lost his keys.* *She said to me, «You are wrong»* → *She said I was wrong.* *They said, «We will help you»* (мне) → *They said they would help me.*

### Reporting verbs — короткий перечень

- **say** — без объекта: *He said (that)...*. Нельзя *«He said me»*.
- **tell** — с объектом обязательно: *He told me (that)...*. Нельзя *«He told that»*.
- **ask** — для вопросов и просьб: *He asked me (where I lived) / (to wait)*.
- **explain to / mention to** — требуют `to + obj`: *He explained to me that...*.
- **suggest** — без объекта, с `V-ing` или `that S should V`: *He suggested going / He suggested that we should go*.
- **advise / recommend / warn / order / urge / beg / remind** — с объектом и `to + V`: *She advised me to wait*.
- **promise / agree / offer / threaten / refuse** — с `to + V`: *He promised to call*.
- **complain / deny / insist** — с `that` или `V-ing`: *He denied taking the money / He denied that he had taken the money*.

<RuleList rules={[
  "<strong>Шаблон:</strong> S + asked (obj) + wh-word (where / when / why / how / what / who / which) + S + V (прямой порядок) + backshift. <em>«Where do you live?» → He asked where I lived.</em>",

### Reported imperatives — приказы, просьбы, советы

<RuleList rules={[
  "<strong>Положительный приказ:</strong> told / asked / ordered / advised + obj + <strong>to + V</strong>. <em>«Wait» → He told me to wait.</em>",
  "<strong>Отрицательный приказ:</strong> told + obj + <strong>not to + V</strong>. <em>«Don't be late» → He told me not to be late.</em> НЕ «told me to don't be late».",

## 9. Мнемоника

<Callout type="teal" title="Лайфхак запоминания">
Запомни простой ритм: **глагол — на шаг назад, обстоятельства — на шаг назад, лица — под рассказчика**. Present → Past, Past → Past Perfect, will → would, can → could, must (долг) → had to. today → that day, tomorrow → the next day, here → there, this → that. I → he/she, my → his/her, you → I/me (если говорят мне). Это работает на 90% случаев. Остальные 10% — исключения (must-дедукция, универсальные истины, уже-backshifted модальные).
</Callout>

## 10. Логика расхождения

<Callout type="blue" title="Почему английский требует backshift">
Backshift — не педантизм и не «согласование ради согласования». Он отражает сдвиг точки отсчёта. Прямая речь живёт в моменте говорения: «I am tired» — это сейчас, относительно момента, когда фраза произносилась. Когда мы пересказываем это спустя время, момент «сейчас» того, кто говорил, уже в прошлом. Английский эту сдвинутую перспективу маркирует в самом глаголе: was tired = «было в момент того высказывания», had lost = «уже было потеряно к моменту того высказывания», would come = «было намерением, направленным в будущее от точки того высказывания». Русскому это не нужно — у нас точка отсчёта плавающая, мы можем пересказывать «изнутри» оригинального момента. Английский же привязывает время глагола к точке репортинга: если она в прошлом — всё в придаточном тоже в прошлом, иначе нарушается грамматическая логика последовательности. Поэтому «He said he is tired» носитель слышит как «он сказал — он сейчас устал», что синтаксически режет ухо: события не в одной плоскости времени.
</Callout>

## 12. Типовые ошибки русскоязычных

| Неверно | Верно | Почему |
| --- | --- | --- |
| He asked where did I live | He asked where I lived | В reported wh-question — прямой порядок, без вспомогательного `did`. |
| He asked if did I have time | He asked if I had time | В reported yes/no question — прямой порядок, без `did`. Backshift: have → had. |
| He told that he was busy | He told me that he was busy | tell требует объект (told me / told us / told her). Без объекта — say. |
| He said me that he was busy | He told me / He said to me | say без `to` не берёт прямой объект. Либо `said to me`, либо `told me`. |
| He said he is tired | He said he was tired | После `said` (Past) — backshift: is → was. Реликт сейчас допустим только для универсальных истин и говорящего сейчас. |
| She said she will come tomorrow | She said she would come the next day | will → would, tomorrow → the next day / the following day. Обстоятельства сдвигаются вместе со временем. |
| He told me to don't be late | He told me not to be late | Отрицательный imperative — `not to + V`, а не `to don't + V`. |
| He said me to wait | He told me to wait / He asked me to wait | Приказ / просьба — через `tell` или `ask` с объектом, не через `say`. |
| She asked if I have finished | She asked if I had finished | После asked (Past) — backshift: have finished → had finished. |
| He said he must to leave (обязанность) | He said he had to leave | must (обязанность) в reported → had to. |
| He said he had to be tired | He said he must be tired | must (дедукция) НЕ сдвигается. Остаётся must в reported. |
| She said she could to swim | She said she could swim | could не сдвигается дальше, и после модального — без `to`. |
| He asked me where do I work | He asked me where I worked | Прямой порядок (where I worked), backshift (work → worked), без `do`. |
| He said me «I am tired» | He said, «I am tired» / He said to me, «I am tired» | В прямой речи `say` без объекта или с `to + obj`. |

## 13. Шпаргалка

| Что преобразовать | Прямая → Reported |
| --- | --- |
| Present Simple | → Past Simple (work → worked) |
| Present Continuous | → Past Continuous (am working → was working) |
| Present Perfect | → Past Perfect (have worked → had worked) |
| Past Simple | → Past Perfect (worked → had worked) |
| Past Continuous | → Past Perfect Continuous (was working → had been working) |
| will | → would |
| going to | → was/were going to |
| can | → could |
| may (возможность) | → might |
| must (обязанность) | → had to |
| must (дедукция) | → must (не меняется) |
| could / would / might / should / ought to | без изменений |
| today / tonight / this morning | → that day / that night / that morning |
| yesterday | → the day before / the previous day |
| tomorrow | → the next day / the following day |
| now | → then |
| ago | → before / earlier |
| here / this / these | → there / that / those |
| I / my | → he/she / his/her (под рассказчика) |
| you (к слушателю) | → I / me / us (под рассказчика) |

| Шаблон | Конструкция |
| --- | --- |
| Statement | S + said (that) + S + V (backshift). tell — только с obj. |
| Yes/no question | S + asked (obj) + if/whether + S + V (прямой порядок, без do, без `?`). |
| Wh-question | S + asked (obj) + wh-word + S + V (прямой порядок, без do, без `?`). |
| Imperative (+) | S + told/asked/advised + obj + to + V. |
| Imperative (−) | S + told/asked + obj + not to + V. |
